# Project description
In this project was create layout of page [lifestyle](https://www.figma.com/design/FXH4IrR8Vho44BpcloBNfc/DEMO-for-Dima-Bukovsky?node-id=1-60652&t=aG39C3oAUu95qKwm-4)
To run project in your computer you need:
1. clone this repository by command:
  ```
  git clone https://github.com/OneMight/Internship-lifestyle.git
  ```
2. Open code editor (I used Visual Studio Code)
3. Open folder with project
4. For best experience you can install extention Live Server
5. If you install Live Server you may run app by clicking the button "Go live" in the right-bottom corner of VS code
6. Open browser and enter in the search string:
   ```
   http://localhost:5500
   ```
